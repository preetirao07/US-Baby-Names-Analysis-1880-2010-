# US-Baby-Names-Analysis-1880-2010
This project analyzes baby name trends in the United States using data released by the Social Security Administration. The dataset contains yearly counts of baby names from 1880 to 2010, making it ideal for exploring long-term patterns in naming popularity.

## Project Analysis Overview

### Popularity & Trend Analysis
  - Visualized how the proportion of babies given a specific name changes over time
  - Calculated the relative rank of any name for any year
  - Identified the most popular names annually
  - Determined which names have increased or declined the most in popularity across decades

### Linguistic & Structural Name Insights
  - Analyzed names based on vowel and consonant patterns
  - Studied how name length has evolved over the years
  - Measured naming diversity to see whether names are becoming more unique
  - Examined changes in first and last letters of names over time
  - Explored how alternate spellings rise or fall in usage

### Cultural & External Trend Analysis
  - Investigated trends in biblical names
  - Analyzed the influence of celebrities and cultural events on name popularity
  - Explored how broader demographic shifts may relate to naming patterns

### Tools & Techniques Used
  - Python (Pandas, NumPy)
  - Data cleaning and transformation
  - Time-series analysis and grouped aggregations
  - Pattern and diversity measurements
  - Matplotlib and Seaborn visualizations
